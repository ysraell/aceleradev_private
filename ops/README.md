My data science Docker image.

Base image: `python:3.7-buster`

## Building:
To build the image:

```bash
./build.sh
```

## Running:
To run the image and set up the servers:

```bash
./run.sh
```

Wait for get the JupterLab URL access.

## Stopping:
To stop the container:

```bash
./stop.sh
```

## Run a shell in the running container: 
To stop the container:

```bash
./exec_bash.sh
```
